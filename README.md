# n8n-nodes-note-me

Automate note creation and reminders with NoteMe in your n8n workflows. Capture notes from various sources, set date-time reminders, organize with tags, and sync everything to your NoteMe mobile app.

[n8n](https://n8n.io/) is a [fair-code licensed](https://docs.n8n.io/sustainable-use-license/) workflow automation platform.

## Installation

Follow the [installation guide](https://docs.n8n.io/integrations/community-nodes/installation/) in the n8n community nodes documentation.

## Operations

### Entry

Create notes and reminders automatically from your n8n workflows:

- **Create Note**: Capture quick notes with text, links, attachments, mentions, and tags
- **Create Reminder**: Set date-time reminders that sync to your NoteMe mobile app
- **Organize**: Add tags to organize and find notes easily
- **Sync**: All notes automatically sync to your NoteMe mobile app

## Credentials

Authenticate with NoteMe using your personal token. Get your token from NoteMe settings to connect this node to your account.

### Getting Your Token

1. Visit [https://docs.qrdyai.com/noteme/n8n-integration](https://docs.qrdyai.com/noteme/n8n-integration) for detailed instructions
2. Log in to NoteMe at [noteme.qrdyai.com](https://noteme.qrdyai.com)
3. Go to settings and copy your token
4. Paste the token into the NoteMe API credentials in n8n
5. The token will be used for all API requests

## Compatibility

Compatible with n8n@1.60.0 or later
